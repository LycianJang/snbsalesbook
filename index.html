📊 CSV / 스프레드시트 데이터 일괄 업로드
📋 CSV 형식 안내
매출 CSV 형식:
날짜,카테고리,항목,결제수단,금액,메모
예: 3,영상촬영,1캠 줌,입금,275000,홍길동

지출 CSV 형식:
날짜,카테고리,금액,메모
예: 1,임대료,880000,3월분

또는 스프레드시트 형식 (자동 인식):
분류,항목,결제,1,2,3,...,31
행에 카테고리/항목/결제수단, 열에 날짜별 금액

✅ 셀 합치기(merge) 된 시트도 OK
빈 칸은 위 행의 분류/항목 값을 자동으로 이어받습니다.
소계, 합계 행은 자동으로 건너뜁니다.

💡 팁: 숫자에 천 단위 쉼표(40,000)가 있으면
CSV 대신 탭으로 분리된 텍스트(.txt/.tsv)로 저장하면 더 안전합니다.
또는 따옴표로 감싸진 CSV("40,000")도 정상 인식됩니다.
📝 메모 일괄 업로드
📋 메모 파일 형식
Google Sheets에서 Apps Script로 추출한 메모 파일:
분류 → 항목 → 결제수단 → 날짜 → 메모
예: 영상촬영   1캠 줌   입금   7   홍길동 결혼식
